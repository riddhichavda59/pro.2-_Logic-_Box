# 📖 Project Description

**Logic Box** is a beginner-friendly Python project that helps students improve their programming and logical thinking skills. The application provides a menu-driven interface where users can generate star patterns, analyze numbers as Even or Odd, and calculate the sum of numbers within a selected range.

This project is ideal for beginners who want practical experience with Python fundamentals.
